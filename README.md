## Overview

**MoltOS** is a Debian-based (12) Linux distribution built with a strong focus on:

- **User Privacy**
- **Defensive Security Analysis**
- **Network Research**
- **Forensics & Data Recovery**
- **Educational Cybersecurity Development**

Unlike typical penetration-testing distributions, MoltOS prioritizes a **clean, modern GNOME experience**, strong default privacy settings, and only legally-safe, open-source, defensive tools.

---

## Security, Privacy & Research Tools

MoltOS provides a carefully curated set of safe, legal cybersecurity and privacy tools.

### 🔹 Privacy Tools
- Tor  
- Torsocks  
- Macchanger  
- DNSCrypt-Proxy  
- UFW Firewall  
- Fail2Ban  
- AppArmor + Profiles  
- WireGuard + wireguard-tools  

### 🔹 Network Analysis
- Wireshark  
- Nmap  
- Masscan  
- Arp-scan  
- DNS Utilities (dig, host, nslookup)  
- Tcpdump  

### 🔹 Web Application Auditing
- Nikto  
- WhatWeb  
- Dirb  
- SQLMap  
- WPScan  
- Wapiti  

### 🔹 Digital Forensics & Recovery
- SleuthKit  
- Binwalk  
- Foremost  
- TestDisk / PhotoRec  
- Bulk Extractor  

### 🔹 Password & File Audit Tools
- John the Ripper  
- Cracklib  
- Wordlists  
- PDFCrack  

### 🔹 Wireless Testing (auditing your own hardware)
- Aircrack-NG  
- Kismet  
- Reaver  

All tools shipped in MoltOS are fully free, open-source, legal, and suitable for **defensive analysis**, **research**, and **education**.

## System Requirements

- **64-bit system (amd64 or x86_64 Arm Family not supported)**  
- **2 GB RAM minimum** (4 GB recommended)  
- **8 GB USB drive or SSD**  
- Works on both **UEFI** and **Legacy BIOS**
- Notice:The system is in early testing phase. Please contact to the MoltOS developers with discord, username is enterteinos and 3rr0r11011.
